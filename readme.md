## Run Instructions ##
1. Download as Zip
2. Extract
3. In IDE of your choice run:
pip install django requests
python manage.py runserver
4. Open this link:
http://127.0.0.1:8000/
5. Follow the prompts to get an API key, it is not saved anywhere so no one is able to access your key, it may take some time to have your API key activated though. If it says it doesn't work just double check, it does work though as I fully tested it.
